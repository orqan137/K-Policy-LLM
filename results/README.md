# Results

이 디렉터리는 공개 가능한 평가 요약 CSV를 제공합니다. 원시 로그, 전체 생성 결과, 대용량 체크포인트는 포함하지 않습니다.

| 하위 디렉터리 | 내용 |
| --- | --- |
| `public_benchmarks/` | KMMLU, CLIcK, KoBEST 등 공개 한국어 지표의 단계별 결과 |
| `internal_eval/` | 프로젝트 내부 생성형 자동채점 평가 결과 |
| `pretraining_selection/` | 15B 토큰학습량 기준 베이스 선택에 사용한 사전학습 체크포인트별 지표 |
