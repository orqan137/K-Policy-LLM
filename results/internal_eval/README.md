# Internal Evaluation

내부 평가는 프로젝트 목표 작업에 맞춰 직접 구성한 803개 생성형 진단 문항을 규칙 기반 자동채점기로 채점한 결과입니다. 사람 평가 점수가 아니며, 범용 모델 순위표로 해석해서는 안 됩니다.

| 파일 | 내용 |
| --- | --- |
| `target_eval_full803_strict_overall.csv` | 네 단계 전체 평균 점수 |
| `target_eval_full803_strict_stage_scores.csv` | 항목별 세로형 결과 |
| `target_eval_full803_strict_stage_scores_wide.csv` | 항목별 가로형 결과 |
