# Configs

이 디렉터리는 K-Policy-LLM 실험에 사용한 주요 설정 파일을 제공합니다.

참고: 설정 파일 일부에는 내부 실험 환경에서 사용한 절대 경로가 남아 있습니다. 공개 저장소에서는 실험 설정과 계보 확인을 위한 참고 자료로 제공하며, 그대로 실행하려면 로컬 환경에 맞게 경로를 수정해야 합니다.

| 하위 디렉터리 | 내용 |
| --- | --- |
| `model/` | 모델과 토크나이저 설정 |
| `train/` | 사전학습, CPT, SFT, DPO 설정 |
| `eval/` | 공개 벤치마크와 체크포인트 평가 설정 |
