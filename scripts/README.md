# Scripts

이 디렉터리는 실험에 사용한 핵심 스크립트 일부를 공개용 참고 자료로 제공합니다.

참고: 스크립트는 원래 내부 실험 환경에서 실행되던 코드입니다. 공개 저장소에서 바로 실행 가능한 완성 패키지라기보다는, 데이터 구성, 학습, 평가, 내보내기, 서빙 절차를 확인하기 위한 reference implementation입니다.

| 하위 디렉터리 | 내용 |
| --- | --- |
| `data/` | 토크나이저 말뭉치, 사전학습 데이터팩 구성 |
| `training/` | 사전학습과 사후학습 실행 참고 스크립트 |
| `evaluation/` | 공개 벤치마크와 내부 생성형 자동채점 평가 |
| `export/` | HF 스타일 export |
| `inference/` | 로컬 추론 예시 |
| `server/` | llama.cpp 실행 예시 |
