# Model Card

이 문서는 K-Policy-LLM의 공개용 모델 카드 초안입니다. 모델 가중치 공개 범위가 확정되면 실제 체크포인트 이름, 배포 형식, 사용 제한, 예시를 보완합니다.

## 모델 개요

- 모델명: K-Policy-LLM
- 모델 유형: dense decoder-only causal language model
- 파라미터 규모: 약 1.221B
- 문맥 길이: 4,096 tokens
- 토크나이저: 한국어 중심 SentencePiece BPE, 80K vocabulary
- 주 사용 영역: 한국 정치, 정책, 법률 문서 이해와 구조화

## 의도한 사용

- 회의록, 법률안, 검토보고서, 정책자료, 공약자료의 요약과 구조화
- 자료 안에서 확인 가능한 근거 기반 질의응답
- 정책 쟁점, 법적 검토 포인트, 수치 변화의 설명 보조

## 사용해서는 안 되는 경우

- 법률 자문, 선거 전략, 정책 판단의 최종 근거
- 공식 출처 확인 없이 현재 사실을 단정하는 사용
- 특정 정당, 후보, 입장을 지지하도록 설득하는 사용

## 알려진 한계

- 공개 객관식 벤치마크 절대 점수는 낮은 수준입니다.
- 표 수치 해석과 복잡한 산술 추론은 안정적이지 않습니다.
- 내부 생성형 평가는 규칙 기반 자동채점이며 사람 평가 점수가 아닙니다.
- 현재 결과는 공개 한국어 특화 LLM과의 경쟁 성능을 주장하기 위한 것이 아닙니다.
