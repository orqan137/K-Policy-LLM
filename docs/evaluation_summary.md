## 해석 기준

- 공개 벤치마크 절대 점수는 낮은 수준입니다.
- KoBEST 일부 2지선다형 지표의 50% 안팎 결과는 무작위 기준선에 가깝게 해석해야 합니다.
- 내부 생성형 평가는 규칙 기반 자동채점이며 사람 평가가 아닙니다.
- 사후학습의 성과는 객관식 점수 상승보다 반복 억제, 직접 답변, 근거 한계 표현, 형식 안정화 사례에서 관찰되었습니다.
