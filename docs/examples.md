# Examples

이 문서는 K-Policy-LLM의 공개용 예시를 정리하기 위한 초안입니다. 실제 모델 파일과 추론 환경이 정리되면 입력 프롬프트와 출력 예시를 추가합니다.

## 예시 유형

- 짧은 사실 질의
- 현재 사실과 기준일 확인
- 회의록 구조화
- 법률안 검토보고서 해설
- 정책·공약 비교
- 근거 부족 상황의 제한 답변
- 표 수치 해석

## 작성 원칙

- 질문, 시스템 지시, 모델 출력, 해석을 함께 제공합니다.
- 기준답변과 모델 출력이 지나치게 동일한 사례는 과적합처럼 보일 수 있으므로 본문 예시로 사용할 때 설명을 함께 붙입니다.
- 성공 사례뿐 아니라 실패 사례와 한계도 함께 제시합니다.
- 현재 사실은 기준일과 출처 한계를 명시합니다.

## 추가할 내용

- SFT로 개선된 단답 사례
- SFT로 개선된 계산 교정 사례
- DPO로 개선된 정책 설명 사례
- DPO로 개선된 근거 한계 표현 사례
- 여전히 실패하는 수치 해석 사례
