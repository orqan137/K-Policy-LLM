# Training Summary

이 문서는 K-Policy-LLM의 학습 과정을 공개용으로 요약한 문서입니다.

## 전체 흐름

```text
사전학습 데이터 구성
        |
80K 한국어 중심 토크나이저 학습
        |
1.221B decoder-only LM 사전학습
        |
15B 토큰학습량 지점 선택
        |
추가 사전학습(CPT)
        |
지도학습(SFT)
        |
선호학습(DPO)
```

## 사전학습

사전학습은 다음 토큰 예측 방식으로 진행했습니다. 최종 데이터팩은 31.21B tokens 규모였지만, 중간 체크포인트 평가 결과를 기준으로 15B 토큰학습량 지점을 사전학습 베이스라인으로 선택했습니다.

| 토큰학습량 | 한국어 확장 가중 평균 |
| ---: | ---: |
| 1.5B | 26.00 |
| 6B | 28.06 |
| 9.5B | 28.06 |
| 13.5B | 28.06 |
| 15B | 28.59 |
| 16B | 27.52 |

## 사후학습

| 단계 | 목적 |
| --- | --- |
| CPT | 법령·정책 문체와 도메인 용어 추가 노출 |
| SFT | 답변 형식, 단답, 문서 구조화, 법률안 해설 학습 |
| DPO | 반복, 질문 복사, 근거 없는 단정 억제 |

- 학습 설정 요약

- 학습 비용과 시간
- 주요 실패와 조정 기록
