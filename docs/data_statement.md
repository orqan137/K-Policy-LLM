# Data Statement

이 문서는 K-Policy-LLM 학습에 사용한 데이터의 공개용 설명서 초안입니다. 원시 데이터 전체를 공개하는 문서가 아니라, 데이터 구성 원칙, 출처 유형, 필터링 방식, 배포 제한을 설명하기 위한 문서입니다.

## 데이터 구성 원칙

- 한국어 일반 언어 능력을 확보하기 위한 대규모 한국어 말뭉치 사용
- 정치·정책·법률 도메인 자료 보강
- 개인정보, 깨진 인코딩, 중복 문서, 품질 낮은 문서 필터링
- 사전학습과 사후학습의 역할 분리

## 주요 원천 유형

- 한국어 웹 말뭉치
- 국회 회의록 기반 자료
- 법률안 검토보고서 요약 자료
- 법령 및 공공 정책 자료
- 선거 정책·공약 자료
- 내부 검토를 거친 합성 지시응답 데이터

## 사전학습 데이터 요약

| 항목 | 수치 |
| --- | ---: |
| 최종 사전학습 데이터팩 | 31.21B tokens |
| 한국어 토큰 비율 | 93.67% |
| 영어 토큰 비율 | 6.33% |
| 최종 기준 베이스 | 15B 토큰학습량 |

## 공개 제한

원천 데이터의 라이선스, 개인정보 위험, 내부 생성 데이터의 공개 가능 범위를 확인한 뒤 공개 범위를 결정합니다. 이 저장소에는 공개 가능한 요약과 재현 가능한 처리 절차를 우선 정리합니다.

## 추가할 내용

- 데이터 출처별 라이선스 확인표
- 전처리 규칙
- 필터링 기준
- 데이터 카드
- 공개 가능한 샘플
