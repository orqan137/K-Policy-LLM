# Improvement Cases

이 디렉터리는 사전학습 베이스라인에서 실패했던 질의가 SFT 또는 DPO 이후 어떻게 바뀌었는지 보여주는 사례를 제공합니다.

| 파일 | 내용 |
| --- | --- |
| `selected_improvement_cases.md` | 대표 개선 사례 |
| `selected_improvement_cases_expanded.md` | 확장 개선 후보와 해석 |
| `posttraining_stage_cases.md` | 사후학습 단계별 사례 보고서 |
| `cumulative_transition_cases.md` | 누적 전이 사례 보고서 |
| `cumulative_transition_case_review.md` | 사례 검산 결과 |
| `report_ready_case_and_benchmark_summary.md` | 보고서용 사례·벤치마크 요약 |

참고: 일부 원본 사례 문서는 실험 검산용으로 작성되어 내부 run label이나 후보 ID를 포함합니다. 공개 발표나 논문 본문에는 사람이 읽기 쉬운 설명으로 다시 정리해 사용하는 편이 좋습니다.
